# Módulo de Transferencias

Este módulo gestiona todas las transferencias de inventario entre bodegas.

## Estructura

- `pages/` - Componentes de página
- `components/` - Componentes reutilizables
- `hooks/` - Hooks personalizados
- `services/` - Servicios de API
- `state/` - Gestión de estado (Zustand, Context, etc.)
- `types/` - Definiciones de tipos TypeScript
- `utils/` - Funciones utilitarias
- `constants/` - Constantes del módulo
- `mocks/` - Datos de prueba
- `schemas/` - Esquemas de validación (Zod, etc.)

## Características principales

- [ ] Listar transferencias
- [ ] Crear nueva transferencia
- [ ] Editar transferencia
- [ ] Eliminar transferencia
- [ ] Aprobar/Rechazar transferencia
- [ ] Generar reportes
- [ ] Buscar y filtrar transferencias
